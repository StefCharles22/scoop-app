About this app

The Scoop web app is a news source site where the user can read the current breaking news, customize their newsfeed and have access to headlines, article title and image associated with feed.

Technology used:
  Backend
    - Ruby on Rails
    - SQlite
    - Devise
    - Paperclip
    - News APi

  Frontend
    - HTML
    - SCSS
    - Bootstrap
